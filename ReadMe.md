# Atividade JS

Please, follow the next steps:
```css
/* Configure o seu usuário e email */
$ git config user.name "Nome Sobrenome"
$ git config user.email "nomesobrenome@email.com"

/* Clonar o repositório remoto para o seu computador local. */
$ git clone https://gitlab.com/InfoJrUFBA/projetos/internos/Pluginfo/Trainee/2020-2/atividade-js.git

/* Crie a sua branch */
$ git checkout -b "minha-nova-branch"

/* Adicione as alterações feitas */
$ git add . /* Adicionará todas as alterações */
$ git add "Arquivo.js" /* Adicionará apenas este arquivo */

/* Commite as alterações adicionadas */
$ git commit -m "Criando Arquivo.js"

/* IMPORTANTE! Antes de subir, confira. */
$ git pull origin master /* IMPORTANTE! Qualquer projeto. */
$ git pull origin minha-nova-branch /* Projetos em grupo na mesma branch. */

/* Suba as alterações commitadas */
$ git push origin minha-nova-branch /* Subirá para o repositório remoto */


/* EXTRAS */

/* Verifique as alterações feitas e onde estão */
$ git status

/* Altere o nome da sua branch */
$ git branch -m nome-antigo novo-nome

/* Abra o VSCode dentro da pasta do seu projeto */
$ code .
```
